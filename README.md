# Data Job Market Analytics via SQL

## 📌 Project Overview
This repository contains a comprehensive SQL-based analytical project designed to explore global tech job market trends. Utilizing a dataset of real-world job postings, this project applies advanced querying techniques to uncover critical insights regarding demand, salary benchmarks, and optimal skill combinations for data professionals.

The main objective is to transition raw, unorganized job market records into actionable career intelligence.

## ❓ Business Questions Answered
The SQL scripts in this repository are structured to solve specific market questions:
1. **Top-Paying Roles:** What are the highest-paying job listings and roles within the data analytics domain?
2. **Skill Demand:** Which technical skills are most frequently requested by employers for data roles?
3. **Salary Alignment:** Which specific skills are correlated with the highest financial compensation?
4. **Optimal Skills:** What are the most optimal skills to learn (intersecting both high demand and high pay)?

## 🛠️ SQL Tech Stack & Advanced Techniques
The project is built on top of **PostgreSQL** and demonstrates proficiency in:
* **Advanced Data Aggregation:** Utilizing `GROUP BY`, `HAVING`, and built-in mathematical/statistical functions to analyze salary spreads.
* **Complex Relational Queries:** Enforcing multi-table `JOIN` statements to connect jobs, companies, and required skill tables.
* **Common Table Expressions (CTEs):** Writing clean, modular, and optimized SQL code via `WITH` clauses for multi-stage data transformations.
